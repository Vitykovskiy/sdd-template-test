# Артефакты glossary

## Назначение
Фиксирует термины и их бизнес-определения в одном контексте.

## Единица декомпозиции
Один файл = один glossary для одного домена или bounded context.

## Критерии разделения
- Разделяй по разному домену, bounded context или владельцу смысла.
- Если термин меняет смысл в другом контексте, выноси его в отдельный файл.
- Не смешивай в одном файле термины из независимых предметных областей.

## Create vs update
- Если канонический файл уже есть и контекст не изменился, обновляй его.
- Если появился новый домен или bounded context, создай новый файл в `docs/business/glossary/`.

## Минимальная готовность артефакта
- Есть сам термин.
- Есть однозначное определение.
- Указан контекст употребления, если без него возможна неоднозначность.
