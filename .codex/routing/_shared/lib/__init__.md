# __init__.py

## Назначение

Файл помечает каталог `lib` как Python package и задаёт краткое текстовое описание пакета общих helper-функций.

## Роль в архитектуре

- делает импорт `from lib.common import ...` корректным после добавления `_shared` в `sys.path`;
- фиксирует, что пакет `lib` предназначен для общих helper-компонентов routing-скриптов.

## Содержимое

Файл не содержит исполняемой логики, кроме module docstring.

## Важные инварианты

- Файл должен оставаться минимальным;
- бизнес-логика и инфраструктурная логика должны находиться в `common.py`, а не в `__init__.py`.
